# Landing Page (Vite + React)

## Setup

```bash
npm install
npm run dev
```

## Link do aplicativo

Por padrão, os botões "Começar" levam para `https://app.chordchartpro.com`.

Para trocar o destino, defina a variável:

```bash
VITE_APP_URL=https://app.chordchartpro.com
```

Em Cloudflare Pages, configure `VITE_APP_URL` em Settings → Environment variables (Production).
